# Responsible Use

This project generates lottery number sets and provides convenience workflows for personal use. It must not be used or described as a way to predict winning numbers.

## No Winning Guarantee

Lottery results are random. Statistical frequency, hot/cold number lists, sum ranges, and weighted generation strategies do not improve the mathematical odds of winning.

Generated numbers are references only. They are not financial advice, betting advice, or predictions.

## Spending Limits

Use this project only within a budget you can afford to lose. Do not use automation to increase lottery spending, chase losses, or bypass your own purchase limits.

Before buying, manually confirm:

- The number of tickets.
- The total purchase amount.
- The draw type and draw date.
- The selected numbers or groups.

## Account Safety

Purchase-related features require account credentials for the lottery service. Use them only on a trusted local machine.

Do not:

- Commit credentials to Git.
- Share credential files.
- Run the app on a public or shared host with real credentials.
- Expose the backend to the internet without authentication and security review.

## Service Terms

External lottery services may change their website, APIs, rules, or terms. You are responsible for using this project in a way that complies with the service rules and applicable law.

## Project Scope

This repository is an open source tool for learning, personal workflow automation, and transparent maintenance. It is not a gambling recommendation system.
